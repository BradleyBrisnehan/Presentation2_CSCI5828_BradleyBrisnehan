Basic Tutorial

From the main screen of the Requirements Management console, select Modules > Create Module. A splash for Create Artifact will open and the specifics can be filled in. If a name is not indicated, it will be assigned the sequential number as the name.

Once a new module is created, the checkmark at the bottom of the Create Artifact screen will automatically open the artifact screen. The right-hand panel has basic module information and control. In the Module Links panel select "Add Link to Artifact" to create links between modules. To give it a try, select "Link To" and select another module from the recent artifacts panel. When you select OK, the link will be created. On the Artifact screen, select "Links Explorer" to graphically view the new link.

Now go to Modules > Browse Modules and select a module with a link present. On the artifact screen, select "Edit" on the upper corner of the right-hand plane. Add text to the Description field and Save the changes.

From the Browse Modules screen, select "Views" from the left-hand plane and save the table as a new View by selecting "Save as New View." Columns can be added by selecting "Configure Page Settings" to change columns in the view and save the changes accordingly.

This tutorial is intended to be a very high-level exposire to the software interface. Comfort with the software will come after hours of exploration on a true project build.
