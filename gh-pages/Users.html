Users

To control collaboration and allow separation of responsibilities, DOORS allows the definition of user roles. The following standard user roles exist:

  • Standard Users can create and modify objects within modules. These users cannot archive data, create new users or user groups, or assign users to a specific group.
  • Project Managers can partition and archive data, create new user groups, and assign or remove users from a specific user group. These users cannot create new users.
  • Database Managers can do everything a Project Manager can in addition to creating new projects and users.
  • Custom Users can have any combination of capabilities as defined by the Database Manager. This allows group responsibilities specific to the current project or business area.

Segregation into groups, allow the development of tailored views so that specific users only see pertinent information in a clean / concise manner.
