Putting It Together

Now that we have consider modules, links, and change tracking we can visualize the overall system model. The strength of DOORS is due to the fact that each user only needs to interact with one object or link at a time. The system model is maintained behind the scenes and, as we will see, allows different users to view differing levels of detail.

What is a system without the user? As with any operating system, DOORS allows the definition of unlimited user groups. Each user group will only view and modify the portions of this model which are meaningful to them.
